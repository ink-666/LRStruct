## 📋 Описание

Лабораторная работа демонстрирует работу со структурами (struct) в C# и перегрузку методов.

### Демонстрируемые концепции:
- Создание и использование структур
- Конструкторы структур (с параметрами и без)
- Статические поля в структурах
- Перегрузка методов
- Обобщенные методы со структурами
- Различия между классами и структурами

## 🚀 Запуск

```
dotnet run
```
